# 7 RINGS - Image Mapping & Upload Guide

We have set up a **self-healing, double-layered image fallback system** inside your website's events section and media gallery! 

Your app is configured to load **your actual local images** first. If a local image file is missing, the website instantly loads a fallback placeholder so that the design remains complete and gorgeous, with no broken image links.

---

### 📂 How to Upload Your Images
You can load these photos directly into the application without needing Google Drive!

1. Open the **File Explorer** in the AI Studio editor sidebar on the left.
2. Create an `images` folder inside the `public` directory (so you have `public/images/`).
3. **Drag and drop** your files from your computer straight into that `public/images/` folder!
4. **Rename** the files to match the exact names in the table below. (For example, rename your Kallon Sallah flyer to `sallah_joint_flyer.jpg`).
